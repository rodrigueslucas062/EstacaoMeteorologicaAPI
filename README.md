# Sobre o projeto

Projeto desenvolvido para fins didáticos, com o objetivo de entender como funciona uma API Web contendo o back-end e front-end utilizando a arquitetura MVC. Sendo assim, o desafio proposto era  criar uma API REST que forneça os dados no formato json, para gerenciar estações meteorologicas.

# Requisitos Funcionais 

- [x] **Obter todas as estações meteorológicas**
- [x] **Obter um aeroporto pelo código aeroportuário IATA**
- [x] **Adicionar uma nova estação**
- [x] **Atualizar uma estação existente**
- [x] **Excluir uma estação**
